# Glow-Up To Do List
A website to outline all the steps for a major glow-up! As well as tracking features to know when you've started a task and completed it. 
## Installation
Using the github in browser, you can open this file by clicking on the project "macadams_katrina_hw3", clicking "code" and then "open in GitHub desktop" this will prompt you to clone the file onto your desktop. 
## Usage
This project file ised to build the Glow-Up To Do List website.
